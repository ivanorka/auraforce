# FiZone Reference Notes

Reference URL: https://fizone.orka.solutions/

Date: 2026-06-05

## What Is Transferable To AuraForce

FiZone is useful as a product/landing structure reference, not as a visual style reference.

Patterns worth carrying into AuraForce:

- visible PDF deck CTA
- web page as live conversation narrative
- PDF as versioned investor/client handoff artifact
- clear "materials still to prepare" logic
- simple final CTA for investor or partner conversations
- focused sections instead of a huge generic feature dump

## What Not To Copy

- language switcher is not needed for AuraForce right now
- FiZone visual identity should not be copied
- AuraForce should stay premium Dubai AI SaaS, darker, sharper and more investor-facing

## Implemented In AuraForce Web Prototype

- header `PDF v0.2` button
- hero `View Product` CTA
- `Investor materials` section
- PDF card with raise and pilot metrics
- product gallery with desktop / mobile app switcher
- data-room / product-proof checklist
- investor / partner CTA section
