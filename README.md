# AuraForce AI Dubai

Working repository for the AuraForce AI investment, product and web materials.
Planned GitHub repository slug: `auraforce`.

AuraForce AI is a premium Dubai-rooted AI command platform for modern businesses. The current fundraising thesis is to start with real estate, prove paid demand around instant lead response and viewing booking, then expand the same platform core into adjacent verticals through installable AI employees.

## Current Strategic Direction

- Build a focused real estate AI sales MVP first.
- Position the product as a turnkey AI command platform, not a chatbot or CRM skin.
- Package capabilities as AI employees with clear roles, permissions, scripts, success metrics and human handoff.
- Keep usage metered from day one: platform subscription, setup fee and usage revenue.
- Use the first raise to build MVP, run Dubai pilots and create commercial proof.

## First Fundraising Target

Recommended initial raise: `$350k-$500k`.

This is enough to build the MVP, run 3-5 Dubai real estate pilots, cover AI/infra buffer and avoid stopping before commercial proof.

There is also a leaner build conversation where a narrower MVP could be estimated at `$75k-$150k`, especially with offshore execution and heavy AI-assisted development. For investor materials, the safer story is to show that lean route as possible but recommend a better-capitalized `$350k-$500k` raise to avoid stopping before pilots, usage economics and commercial proof.

## Working Rule

Every new material should strengthen one of these proof points:

1. The problem is expensive and urgent.
2. The first wedge is narrow enough to ship.
3. The product can expand into a larger platform.
4. The business model protects margin.
5. The team has a credible path to execution.
