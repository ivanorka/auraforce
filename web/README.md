# AuraForce AI Landing Prototype

This is a static HTML/CSS landing page prototype for investor and early client conversations.

Open through the local preview server:

- Landing page: `http://127.0.0.1:5174/web/index.html`
- Product screens: `http://127.0.0.1:5174/web/screens.html`
- PDF deck source page: `http://127.0.0.1:5174/web/pitch-v0-2.html`

The page uses the source pitch structure:

- problem
- solution
- real estate wedge
- AI employees
- business model
- roadmap
- investment logic

Future updates should keep product screens focused on investable proof, not every possible module.
